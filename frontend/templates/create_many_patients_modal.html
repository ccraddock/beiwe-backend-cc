Add many new patients

Create a number of new Patient IDs, and download a CSV file of the new Patient IDs with their registration passwords.

Important: there is no way to delete a Patient ID! Make sure you really do want to create this many new Patient IDs.

This may take several minutes, because it's computationally intensive for Beiwe to create cryptographically secure keys for each patient. Do not close this window until your file finishes downloading.
